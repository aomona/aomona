## About me

I build experimental products where **AI meets real-world workflows**—from adaptive task agents and realtime cooking assistance to developer tools and Minecraft automation.

- 🍐 Working on **PEAR Agent**, a runtime that plans, executes, assesses, and replans
- 🧠 Interested in agent architecture, realtime voice, resilient state, and human-in-the-loop UX
- 🛠️ Also exploring WebGL, Rust, NixOS, Minecraft tooling, and the open web
- 🌏 Based in Japan
